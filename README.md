# 💳 Sistem KantinPay ITK

Proyek ini adalah sistem pembayaran digital sederhana untuk kantin di **Institut Teknologi Kalimantan (ITK)**. Sistem ini dirancang menggunakan paradigma **Object-Oriented Programming (OOP)** dengan bahasa pemrograman **Kotlin**.

---

### 🚀 Fitur Utama
* **Enkapsulasi Data:** Melindungi data sensitif seperti Saldo Mahasiswa, PIN, dan Stok Menu menggunakan modifier `private`.
* **Validasi Transaksi:** Memastikan transaksi hanya terjadi jika PIN benar, stok tersedia, dan saldo mencukupi.
* **Manajemen Stok Otomatis:** Stok makanan akan berkurang secara otomatis setiap kali transaksi berhasil.

### 📂 Struktur Class
1.  **Mahasiswa:** Mengelola data identitas, saldo, dan verifikasi PIN.
2.  **Menu:** Mengelola informasi makanan, harga, dan ketersediaan stok.
3.  **KantinPay:** Sebagai pengontrol logika bisnis dan proses transaksi.
